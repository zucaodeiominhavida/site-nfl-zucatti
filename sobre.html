Sobre o Projeto NFL Today

Este site foi desenvolvido como parte do projeto da disciplina de Pensamento Computacional do 3º ano do Ensino Médio.

Objetivo

Mostrar os principais jogadores da NFL em 2023, com suas estatísticas e conquistas mais recentes.

Critérios

Os jogadores foram selecionados com base em seu desempenho na temporada 2022-2023 e impacto em seus times.

Tecnologias

  • HTML5 semântico
  • CSS3 para estilização
  • JavaScript para interatividade
  • GitHub Pages para hospedagem
